Every run number has the same metadata. Each run (1, 2 and 3) has been performed 5 times. The runs are labelled as SM Run 1_1.csv (for example) this is the first replicate of run 1. The numbering after that keeps on going. Note that the same materials were used in all runs. 

To get the growth parameters use the files growthparameter_calculations_Plotting_Runi_j.R where I is the run and j the replica. This outputs an image with the overview of the plate, the fits for each well and the background. It outputs a csv with the growth parameters.

All the data is in the 'Rawdata' folder and all the output is large and for the moment not added to the repository=. The output csv files are available in the output_data folder.
